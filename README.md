## How to run
1. Make sure you have Java installed.
2. Compile the code:
   ```bash
   javac TicTacToe.java
   
3. Run the Programm:
  java TicTacToe
  
4. How to play
   The board is 3×3.
   Player X starts first, then Player O.
   Enter row (1-3) and column (1-3) when prompted.
   The game will tell you if someone wins or if it’s a draw.
  
5. Code structure
  initializeBoard() — prepares the empty board.
  printBoard() — shows the current board.
  playerMove() — reads and validates the move.
  hasWon(player) — checks if that player has three in a row.
  isBoardFull() — checks for a draw.
